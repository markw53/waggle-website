# Waggle - Dog Breeding Platform

## Features

- User authentication
- Dog registration with health verification
- Admin approval system
- Messaging system
- Match requests

## Tech Stack

- React + TypeScript
- Firebase (Auth, Firestore, Storage)
- Tailwind CSS
- Vite

## Setup

1. Clone repo
2. `npm install`
3. Create `.env` file
4. `npm run dev`

## Deployment

`npm run build`
